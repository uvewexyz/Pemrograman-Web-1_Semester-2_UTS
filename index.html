Tentang Saya

Halo! Saya adalah seorang Jr. Cloud Engineer di PT Boer Technology. Saat ini saya sudah memiliki 2 tahun pengalaman dalam mengelola virtual private cloud. Platform infrastruktur yang saya kelola adalah OpenStack dan terintegrasi dengan Ceph sebagai backend storage-nya. Selain itu, saya juga pernah menggunakan GitLab sebagai CI/CD dan Ansible sebagai automasi untuk mempermudah pekerjaan.

Saya memiliki minat untuk mendalami dunia DevOps dan saat ini sedang mempelajari dunia containerisasi seperti docker dan podman. Saya percaya sebuah quotes dari CR7 alias Ronaldo yaitu "practice makes perfect" yang saya jadikan keyakinan bahwa dedikasi saya dalam mempelajari DevOps tanpa menyerah diperlukan agar bisa mengembangkan potensi diri saya.
